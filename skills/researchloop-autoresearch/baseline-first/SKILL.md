# Baseline First

Read the shared core loop reference: `../references/core-loop.md`.

Use this skill when the repo has no trustworthy baseline yet.

- identify the baseline command and metric
- keep the first run as cheap as possible
- write the baseline into `.researchloop/goal.md` or `.researchloop/plan.md`
- do not optimize until the baseline exists
