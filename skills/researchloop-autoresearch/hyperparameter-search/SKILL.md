# Hyperparameter Search

Read `../references/search-playbook.md` and `../references/core-loop.md`.

Use this skill for small, ranked searches over trainable knobs.

- change one knob family at a time
- use a narrow candidate list
- record the grid and the best run
- stop when the search stops paying for itself
