## What They Do

Promptfoo focuses on testing prompts, models, agents, and RAG systems. It is CLI-first, local-first, and built around automated evaluation and red teaming.

Useful traits:

- declarative config
- CI/CD support
- model comparison
- local/private evaluation
- security and vulnerability reports

## What To Learn

- Evaluation should be easy to run before the user trusts the system.
- CLI + config is the right shape for repeatable checks.
- Local-first matters for trust and privacy.
- Comparisons should be easy to read, not hidden in a notebook.

## How ResearchLoop Differs

- ResearchLoop is about autonomous research loops, not just prompt tests.
- It needs durable state, run ledgers, and agent instructions.
- Prompt evaluation should be one adapter or subcommand, not the whole product.
