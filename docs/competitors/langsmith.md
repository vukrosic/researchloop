# LangSmith

Sources:

- [LangSmith evaluation page](https://www.langchain.com/evaluation)
- [LangSmith evaluation concepts](https://docs.langchain.com/langsmith/evaluation-concepts?mode=ui)

## What They Do

LangSmith covers evaluation across the lifecycle: offline benchmarking, unit/regression tests, online monitoring, heuristic checks, LLM-as-judge, pairwise comparisons, and human annotation queues.

## What To Learn

- Split execution from evaluation.
- Support both automated scoring and human review.
- Give users a way to compare variants systematically.
- Treat evaluation as a workflow, not a one-off script.

## How ResearchLoop Differs

- ResearchLoop is narrower and more file-based.
- It should install into any repo and generate the workflow skeleton.
- It can borrow the evaluation ideas, but not the full platform surface.
