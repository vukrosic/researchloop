# Users And Customer Discovery

This folder tracks people who might use ResearchLoop.

The goal is not to sell too early. The goal is to learn how real researchers run experiments and where the agent harness saves time.

## Segments

- PhD students
- AI research labs
- independent researchers
- companies optimizing prompts, agents, evals, models, or fine-tunes

## Weekly Target

- 3 short messages to potential users
- 1 deeper conversation or repo walkthrough every two weeks
- 1 written learning note per week

## What To Ask For

Ask for a repo walkthrough, not generic opinions.

Useful ask:

`Can I watch how you currently plan, run, and track experiments in one repo? I am building an open source npm harness that helps Codex/Claude run disciplined research loops.`
